
---------------구글 폰트 적용----------------

1. 구글에 "구글 폰트" 검색 
2. google fonts 라는 사이트 접속
3. 왼쪽 language 에서 korean 검색하여 체크
4. black han sans 라는 글꼴을 예제로 사용
5. 원하는 글꼴을 클릭하면 우측 상단에 get fonts 라는 버튼을 클릭
6. 우측에 get embed code 를 클릭
7. embed code in the <head> of your html 에 있는 코드를 복사
	html 파일에 head 태그 안에 넣어준다.
8. 위에 복사한 코드 아래에 CSS class 라고 적힌 코드를 복사
	css 파일에 붙여넣기 한 후에 선택자를 바꿔준다.

---------------아이콘 폰트 적용----------------

1. 구글에 "폰트어썸 cdn" 을 검색
2. cdnjs 소속의 font-awqsome - Libraries 라는 사이트 접속
3. 주황색으로 표시된 url 을 두번째 아이콘을 클릭해서 복사
4. html 파일의 <head> 태그 안에 붙여넣기 - 아이콘 폰트 사용 준비 완료
5. 구글에 "폰트어썸" 을 검색
6. 파란색 깃발이 있는 font awesome 사이트 접속
7. 화면 중앙 하단에 61,908 Icons 를 클릭
8. 원하는 아이콘을 클릭해서 우측에 있는 코드를 클릭하여 복사
9. html 파일의 <body> 태그에 붙여넣기를 하면 아이콘이 출력됨
